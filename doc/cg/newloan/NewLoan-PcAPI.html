1.2. 接口规范

1.2.1. 接口格式
1.接口均采用 Form-data 格式
2.需要鉴权的接口，请在 Headers 中添加 Authorization : "Bearer 鉴权token"
1.2.2. HTTP错误响应统一示例
{
  "code" : 0, // 0：成功
                -1：失败
                -3：异常
  "msg" : "提示信息",
  "data" : "请求返回数据"
}
